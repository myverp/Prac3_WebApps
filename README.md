# Звіт до практичної роботи 3

## Тема

Розробка вебзастосунку для оцінки економічного ефекту від покращення точності прогнозування потужності сонячної електростанції.

## Вимоги із завдання

- визначити частку потужності в допустимому інтервалі за нормальним розподілом;
- порівняти результати до і після покращення прогнозу;
- обчислити дохід, штраф і чистий прибуток;
- відтворити контрольний приклад із PDF;
- оформити сторінку у простому й зрозумілому вигляді.

## Програмна реалізація

- застосунок реалізовано мовою Go;
- для обробки параметрів використано `net/http`;
- серверний рендер сторінки побудовано на `html/template`;
- імовірність у допустимому інтервалі визначається через функцію нормального розподілу;
- результати двох сценаріїв розраховуються на сервері без JavaScript;
- підсумок виводиться у вигляді метрик та порівняльної таблиці.

## Результат роботи

У ході виконання практичної роботи створено вебзастосунок, який дозволяє змінювати середню потужність, ціну електроенергії, допустиме відхилення та значення сигми для двох сценаріїв. Після обчислення сторінка відображає допустиму частку потужності, дохід, штраф та чистий прибуток до і після покращення прогнозу.

## Висновок

Розроблений Go-застосунок відповідає вимогам завдання та наочно демонструє вплив точності прогнозування на економічний результат сонячної електростанції. Реалізація є простою, читабельною та зручною для перевірки контрольного прикладу.
